# Shoe Investments 2.0

Herpositionering van de Shoe Investments website: van "de meest ondernemende investeringsmaatschappij van Nederland" naar "A Dutch family office with global venture reach."

## Originele versie

De v1 van deze website staat op: [cecedeboon-png/shoe-investments-website](https://github.com/cecedeboon-png/shoe-investments-website)

## Wat is nieuw in 2.0

- **Nieuwe positionering:** International family office met focus op AI-tech via fund-to-fund en SPV's
- **Drie pijler structuur:** Direct Investments, Fund-to-Fund, SPV Follow-on
- **Notable exposure:** Anthropic, Revolut, Airwallex, Leasy, Replit prominent
- **Premium design:** Donker kleurenpalet, minimalistisch, internationale uitstraling
- **Engels als primaire taal**

## Tech stack

Statische HTML/CSS/JS site, gehost via GitHub Pages.
